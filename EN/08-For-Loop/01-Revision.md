# Revision 
Before proceeding further, let's remind ourselves about the program concepts and techniques that we have learned in the last session.

# Nested Conditions
```java
if (condition1) {
    if (condition2)
        // body; 
    else
        // body;
}
```

# Complex Conditions with `&&,` `||`, `!` and `()`
```java
if ((x == left || x == right) && y >= top && y <= bottom) {
    System.out.println(…);
}
```

# Switch-Case Conditions
```java
switch (selector) {
    case value1:
        construction;
        break;
    case value2:
    case value3:
        construction;
        break;
    …
    default:
        construction;
        break;
}
```
