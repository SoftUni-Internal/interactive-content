## Description
Write a program to **read** an integer **n** and **print all triples of the first n small Latin letters**, ordered **alphabetically**.

### Example
| **Input** | **Output** |
| --- | --- |
| 2 | aaa |
| | aab |
| | aba |
| | abb |
| | baa |
| | bab |
| | bba |
| | bbb |
