# Summary

## Nested Conditions
```java
if (condition1) {
    if (condition2)
        // body; 
    else
        // body;
}
```

## Complex Conditions with Logical Operators
```java
if ((x == left || x == right) && y >= top && y <= bottom)
    System.out.println(…);
```

## Switch-Case Conditions
```java
switch (selector) {
    case value1:
        construction;
        break;
    case value2:
    case value3:
        construction;
        break;
    …
    default:
        construction;
        break;
}
```
