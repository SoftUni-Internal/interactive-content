## În această lecție ați învățat:

- Buclele for
  
   * Definirea buclelor for

   * Buclele for repetă un anumit bloc de cod

```java live
for (int i = 1; i <= 10; i++) {
  System.out.println(i);
}
```

- Buclele for cu pas special

``` java live
for (int i = 10; i >= 1; i--) {
  System.out.println(i);
}
```

- iterând peste cuvinte

```java live
for (char ch = 'a'; ch <= 'd'; ch++) {
  System.out.print(ch + " ");
}
```

- Prefixes 

``` java
int i = 1;
      System.out.println(++i);
      System.out.println(i);
```

- Postfixes

``` java
int i = 1;
      System.out.println(i++);
      System.out.println(i);
```
